JavaScript

JavaScript là ngôn ngữ phổ biến bậc nhất trong thế giới web hiện nay. Bạn sẽ được giới thiệu những kiến thức cơ bản để tiến thêm một bước vào thế giới lập trình web, là nền tảng cho khoá học jQuey.

JavaScript & DOM

Khoá học JavaScript & DOM cung cấp cho học viên các kiến thức để viết mã tương tác với từng elements của trang web. Với kiến thức này, các bạn sẽ có nền tảng để học về các framework như Angular.js, React.js và Ember.js

jQuery Cơ Bản

jQuery là khoá học giúp bạn làm cho trang web có khả năng tương tác với người dùng. Khoá học này đi vào nội dung bài học thông qua các ứng dụng mẫu. Giúp bạn tiếp cận một cách thực tế và dễ hiểu.

AJAX & JSON

Bootstrap

Tất cả các kiến thức về HTML, CSS và JavaScript cũng như jQuery đã có thể giúp bạn tự tin xây dựng được các giao diện từ cơ bản đến phức tạp đảm bảo hiển thị tốt trên nhiều kích thước màn hình. Tuy nhiên, chúng sẽ tiêu tốn của chúng ta khá nhiều thời gian và công sức. Bootstrap CSS Framework ra đời chính là giúp chúng ta giải quyết vấn đề đó.

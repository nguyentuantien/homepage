Thiết kế web Điểm xuất phát cho người mới bắt đầu sự nghiệp Web Dev. Từ một file photoshop (ảnh của trang web), bạn sẽ tự tay trích xuất hình ảnh, front chữ, đo đạc... để tạo ra một trang web hiển thị trên Internet, và đó gọi là Web Tĩnh. Bạn có thể tham khảo thành quả của học viên CiOne tại đây .

HTML

HTML là ngôn ngữ đầu tiên bạn cần biết để bước vào con đường lập trình web. Qua khóa học ngắn này, bạn sẽ được học rất kỹ về HTML để có thể vận dụng xây dựng được cấu trúc một trang HTML

Bắt đầu học →

CSS

CSS giúp bạn trang trí tài liệu HTML trở nên bắt mắt. Khóa học này tiếp nối khóa học HTML để giúp bạn hoàn thiện trang HTML trong khóa học trước.

Bắt đầu học →

Layout (Bố cục website)

Khóa học này giúp bạn tổng hợp kiến thức của HTML & CSS để bố cục trang web theo đúng chuẩn HTML5. Từ kiến thức đó, bạn có thể tự làm một website cho riêng mình.

Bắt đầu học →

Photoshop

Khoá học Photoshop cho Developer cung cấp các kiến thức sử dụng Photoshop cơ bản như cắt, tách layout, đo kích thước, màu sắc, font chữ, màu chữ.

Bắt đầu học →

Responsive Design

Responsive Web Design giúp bạn thiết kế giao diện hiển thị tốt trên các thiết bị có kích thước màn hình khác nhau. Đây là một trong những xu hướng thiết kế mà bạn cần phải nắm rõ để tạo ra các website thân thiện với người sử dụng.
